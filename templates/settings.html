Guide de configuration

Zoho Mail

  1. Connectez-vous à mail.zoho.eu
  2. Allez dans Paramètres → Mail Accounts
  3. Activez IMAP Access
  4. Créez un mot de passe d'application si 2FA activé
  5. Copiez vos identifiants dans le fichier .env

Shopify

  1. Allez dans Admin Shopify → Paramètres → Apps
  2. Cliquez sur "Développer des apps"
  3. Créez une nouvelle app
  4. Accordez les permissions : read_orders, read_customers
  5. Installez l'app et copiez l'Access Token

Claude API

  1. Créez un compte sur console.anthropic.com
  2. Ajoutez des crédits (min $5)
  3. Allez dans API Keys
  4. Créez une nouvelle clé API
  5. Copiez-la dans le fichier .env

Règles d'automatisation

Ces règles sont configurées dans le fichier .env

Auto-envoi Suivi de commande

Envoie automatiquement si tracking disponible

AUTO_SEND_TRACKING=true

Auto-envoi Confirmation retour

Envoie automatiquement les confirmations de retour

AUTO_SEND_RETURN_CONFIRM=false

Intervalle de vérification

Fréquence de vérification des nouveaux emails

EMAIL_CHECK_INTERVAL=300
